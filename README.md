# greenlife
App móvil enfocada al cuidado de plantas en estilo comunidad, es decir de tipo ecológico.  GreenLife 🌿, la herramienta perfecta para crear una comunidad de usuarios que gusten de compartir contenido sobre las plantas de su agrado, hasta priorizar su ubicacion, los cuidados, fotos y demas informacion que sea facilmente visible en la aplicacion.
